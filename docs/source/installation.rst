Installation
============

The most recent release can be installed from `PyPI
<https://pypi.org/project/pao_plusplus>`_ with uv:

.. code-block:: console

    $ uv pip install pao_plusplus

or with pip:

.. code-block:: console

    $ python3 -m pip install pao_plusplus

Installing from git
-------------------

The most recent code and data can be installed directly from GitHub with uv:

.. code-block:: console

    $ uv pip install git+https://github.com/elinscott/pao-plusplus.git

or with pip:

.. code-block:: console

    $ python3 -m pip install git+https://github.com/elinscott/pao-plusplus.git

Installing for development
--------------------------

To install in development mode with uv:

.. code-block:: console

    $ git clone git+https://github.com/elinscott/pao-plusplus.git
    $ cd pao-plusplus
    $ uv pip install -e .

or with pip:

.. code-block:: console

    $ python3 -m pip install -e .
